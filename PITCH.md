# Pitch

## Krátké představení
Tento projekt demonstruje, jak z jednoho orchestrace běhu vytvořit a dál rozvíjet statický multi-page web připravený na GitHub Pages.

## Hlavní myšlenka
- Generace webu + strukturované iterace + publish v jednom toku.

## Proč je to užitečné
- Zkracuje cestu od nápadu k veřejně dostupnému webu.
- Udržuje konzistenci mezi stránkami přes manifest a sdílenou navigaci.

## Čím je to zajímavé
- Umí tvořit i aktualizovat existující projekt bez úplného přegenerování.
- Delivery pack usnadňuje demo i předání projektu stakeholderům.
